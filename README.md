# 學生學習成果認證與展示系統

## 系統簡介
這是一個讓學生展示學習成果的平台，所有成果需經過管理員審核才會公開顯示。

## 功能說明

### 學生功能
- 註冊/登入帳號
- 上傳個人照片
- 編輯個人資料（姓名、電子郵件、簡介）
- 新增學習成果（擅長科目、程式語言、參與競賽、取得證照）
- 編輯/刪除「待審核」狀態的成果
- 查看所有自己的成果及審核狀態

### 管理員功能
- 登入後台
- 查看所有學生提交的成果
- 將待審核的成果改為「已認證」或「不通過」
- 依狀態篩選成果列表

### 訪客功能
- 搜尋具備特定技能的學生
- 搜尋結果只顯示「已認證」的成果

## 安裝步驟

### 1. 建立資料庫
1. 開啟 phpMyAdmin 或其他資料庫管理工具
2. 匯入 `database.sql` 檔案
3. 系統會自動建立資料庫和測試帳號

### 2. 設定資料庫連線
編輯 `config.php` 檔案，修改資料庫連線資訊：
\`\`\`php
define('DB_HOST', 'localhost');      // 資料庫主機
define('DB_NAME', 'student_portfolio'); // 資料庫名稱
define('DB_USER', 'root');           // 資料庫使用者
define('DB_PASS', '');               // 資料庫密碼
\`\`\`

### 3. 設定檔案權限
確保 `uploads/` 資料夾具有寫入權限（用於儲存使用者照片）

### 4. 測試帳號
系統已建立以下測試帳號：

**管理員帳號**
- 帳號：admin
- 密碼：admin123

**學生帳號**
- 帳號：student1
- 密碼：student123

## 使用技術
- PHP（PDO資料庫操作）
- MySQL資料庫
- Session管理
- 檔案上傳
- Form表單處理
- 基本的HTML/CSS

## 注意事項
1. 學生只能編輯/刪除「待審核」狀態的成果
2. 已認證或不通過的成果無法修改
3. 訪客搜尋只會顯示「已認證」的成果
4. 照片上傳限制：JPG、PNG、GIF格式
5. 所有密碼使用 `password_hash()` 加密儲存
